# Carla Muniz Advocacia (Next.js 14)

## Rodar localmente
```
npm install
npm run dev
```

## Deploy na Vercel
1) Push no GitHub
2) Vercel → Add New Project → Import
3) Variáveis de ambiente: RESEND_API_KEY / CONTACT_TO
4) Deploy
